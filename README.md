# INTERSTELLAR HIGHWAY <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Telegram-Animated-Emojis/main/Travel%20and%20Places/Rocket.webp" alt="Rocket" width="50" height="50" />

A program created to optimize the best trade routes in space. This is a platform built specifically for smart, scalable routes, a logistics planning platform that will serve as the foundation of a safer, more reliable space trade infrastructure.

## Purpose
As humanity expands its reach out into the stars, the Golden Age of Space Commerce is fully underway.
Interplanetary trade has transformed the way we live, work, and connect; space commerce has become
the backbone of this new modern economy.
To properly support this rapidly growing network; a new initiative has emerged: A spacefaring logistics
platform that can help merchants, cargo fleets, and independent traders plan the safest, and most efficient
routes between planets, and stations. The goal is to build a smart, scalable route planning logistics
platform that will serve as the foundation of a safer, and more reliable space trade infrastructure.

## Instructions on how to run the program:
* First, run the program
* Once inside the program, hit mission
* Set departure and destination
* Set Launch Window
* Select your ship
* Set payload mass
* Once everything is set, hit Launch
* Watch the trade happen in real time
